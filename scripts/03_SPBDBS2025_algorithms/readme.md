\f0\fs24 \cf0 # SPBDBS2025 Algorithms\
\
This directory contains the reproducible numerical experiments accompanying the SPBDBS2025 methodology for Hawkes processes and covariate-Hawkes processes.\
\
The scripts demonstrate the implementation of the three main algorithms together with Monte Carlo calibration studies and a covariate-Hawkes extension.\
\
---\
\
## Directory\
\
```text\
scripts/03_SPBDBS2025_algorithms/\
```\
\
All scripts assume that the repository root is the working directory and that all project functions have been loaded with\
\
```r\
source("R/load_all.R")\
```\
\
Generated outputs are written to\
\
```text\
results/\
figures/\
```\

## Running the complete suite\
\
From the repository root,\
\
```r\

\
scripts <- file.path(\
    "scripts/03_SPBDBS2025_algorithms",\
    c(\
        "26_SPBDBS2025_algorithm1_wald_test.R",\
        "27_SPBDBS2025_algorithm2_corrected_gof.R",\
        "28_SPBDBS2025_algorithm3_equality_wald_test.R",\
        "29_SPBDBS2025_wald_calibration.R",\
        "30_SPBDBS2025_algorithm2_gof_calibration.R",\
        "31_SPBDBS2025_covariate_model_example.R",\
        "32_SPBDBS2025_covariate_wald_calibration.R",\
        "33_SPBDBS2025_covariate_algorithm2_gof.R",\
        "99_collect_SPBDBS2025_results.R"\
    )\

    source(s)\

## Numerical status\
\
Current numerical experiments indicate that\
\
- Algorithm 1 Wald tests are well calibrated.\
- Algorithm 3 equality Wald tests are well calibrated.\
- The subsampled version of Algorithm 2 exhibits the best empirical calibration.\
- The covariate-Hawkes maximum likelihood estimator accurately recovers the simulated parameters.\
- Covariate-Hawkes Wald tests are close to the nominal significance level.\
- Covariate-Hawkes residual diagnostics are consistent with the exponential time-rescaling theorem.\

## Note on Script 33\
\
Script **33_SPBDBS2025_covariate_algorithm2_gof.R** provides residual-based goodness-of-fit diagnostics for the covariate-Hawkes model and executes successfully as part of the complete workflow.\
\
The residual diagnostics are based on the covariate compensator and provide an empirical extension of the SPBDBS goodness-of-fit methodology. The fully derived theoretical correction developed for the classical Hawkes model (Algorithms 27 and 30) has not yet been generalized to the covariate-Hawkes setting. Consequently, the corrected statistic implemented in Script 33 should currently be regarded as an empirical diagnostic rather than a theoretically established extension of Algorithm 2.}
